git常用命令

  • git init

    git 仓库初始化

  • git status

    查看git的状态，可以看那些新增文件或者是已经修改过的文件没有add到本地仓库

  • git add <file>

    将某个文件或文件夹添加到本地仓库

  • git rm --cached <file>

    将某个文件或文件夹从本地仓库删除

  • git reset HEAD <file>

    对已经add到本地仓库的某个修改过的文件或文件夹，恢复到add之前（即撤销修改文件的add）

  • git rm --cached <file>

    将某个文件或文件夹从本地仓库删除

  • git commit -m "引号内写此次提交的简述"

    将所有add的内容提交到本地仓库，并简述此次提交，方便以后的回滚

  • git log

    查看git提交的日记

  • git remote add <远程仓库名> <远程仓库地址>

    添加一个远程仓库

  • git remote add <远程仓库名> <远程仓库地址>

    添加一个远程仓库

  • git push -u <远程仓库名> <本地分支名>

    将本地仓库的内容提交到远程仓库，第一次push时需要加 -u <远程仓库名> <本地分支名>，下次提交可以忽略掉

  • git pull <远程仓库名> <本地分支名>

    从远程仓库pull最新的内容（比如有其他伙伴提交了新内容）到本地

  • git diff HEAD

    比较两个版本的差异

  • git branch <分支名>

    创建一个的分支

  • git checkout <分支名>

    切换到某个分支

  • git merge <分支名>

    合并分支

  • git branch -d <分支名>

    删除分支
