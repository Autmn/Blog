在CentOS下搭建一个ftp服务器

  • 使用yum 安装ftp

    yum -y install ftp vsftpd

  • 进入ftp配置文件

    cd /etc/vsftpd

  • 备份ftp配置文件

    cp vsftpd.conf vsftpd.conf.origin

  • 关闭匿名登录

    vim vsftpd.conf
    将#anonymous_enable=NO的#去掉然后保存退出（：wq)

  • 创建一个ftp文件夹

    mkdir ftpFile

  • 使用adduser命令创建ftp用户

    adduser -d /ftpFile -g ftp ftpuser1

    说明：
    -d ftp文件夹名（用户上传的文件将保到此文件夹)
    -g 用户名所在的用户组
    ftpuse1 为本次创建的用户名

  • 为用户名设置密码 passwd <你的密码>
  • 设置vsftpd开机启动

    systemctl enable vsftpd

  • 重启ftp

    systemctl restart vsftpd
